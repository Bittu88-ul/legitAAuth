IMPORTANT DEPENDENCY NOTICE

If you encounter an error while integrating Auth.cs, please install/update the System.Text.Json dependency using the NuGet Package Manager in Visual Studio.

1

Add Auth.cs to Your Project

First, click the top button to download Auth.cs. Copy this file into your Visual Studio project folder, right-click your project in Solution Explorer, select Add > Existing Item..., and choose Auth.cs.

At the top of your login/form class file, import the namespace:

using Auth; // Load the LegitAuth namespace
2

Initialize in Form Load

Double-click your login form in the Visual Studio designer to create the Form_Load event. Put the initialization block there using your application credentials:

private void LoginForm_Load(object sender, EventArgs e)
{
    // Initialize application credentials
    LegitAuthApp.init(
        _name: "YOUR_APP_NAME",
        _ownerid: "YOUR_OWNER_ID",
        _secret: "YOUR_SECRET",
        _version: "1.0" // Must match application settings version
    );
}
3

User/Password Login

Double-click your Login button to create the click handler. Add the async keyword to your event handler method:

private async void btnLogin_Click(object sender, EventArgs e)
{
    string username = txtUser.Text.Trim();
    string password = txtPass.Text;

    if (string.IsNullOrEmpty(username) || string.IsNullOrEmpty(password))
    {
        lblStatus.Text = "Please fill in all credentials.";
        return;
    }

    // Trigger async login request
    await LegitAuthApp.login(username, password);

    if (LegitAuthApp.response.success)
    {
        MessageBox.Show("Welcome back!", "Success", MessageBoxButtons.OK, MessageBoxIcon.Information);
        
        // Hide login form and open main application panel
        MainAppForm mainPanel = new MainAppForm();
        mainPanel.Show();
        this.Hide();
    }
    else
    {
        // Displays exact reason (e.g. incorrect password, account expired, HWID mismatch)
        lblStatus.Text = LegitAuthApp.response.message;
        Console.Beep(500, 250);
    }
}
4

License-Only Verification

If your application uses single-use license keys directly instead of usernames/passwords, implement this click handler:

private async void btnLoginKey_Click(object sender, EventArgs e)
{
    string key = txtLicenseKey.Text.Trim();

    if (string.IsNullOrEmpty(key)) {
        lblStatus.Text = "Please enter your license key.";
        return;
    }

    await LegitAuthApp.license(key);

    if (LegitAuthApp.response.success) {
        MessageBox.Show("License verified successfully!", "Welcome", MessageBoxButtons.OK, MessageBoxIcon.Information);
        new MainAppForm().Show();
        this.Hide();
    } else {
        lblStatus.Text = LegitAuthApp.response.message;
    }
}

Download C++ Win32 Header SDK

1

Add Auth.hpp to Your Project

Download Auth.hpp using the top button. Place it into your project source folder and include the header file:

#include "Auth.hpp"
2

Initialize and Login

Create an instance of the LegitAuth::Auth class and pass your credentials, then execute login():

#include <iostream>
#include "Auth.hpp"

int main() {
    // Initialize application credentials
    LegitAuth::Auth auth(
        "YOUR_APP_NAME",
        "YOUR_OWNER_ID",
        "YOUR_SECRET",
        "1.0"
    );

    std::string username, password;
    std::cout << "Username: ";
    std::cin >> username;
    std::cout << "Password: ";
    std::cin >> password;

    if (auth.login(username, password)) {
        std::cout << "Welcome to the application!\n";
        std::cout << "Dev Message: " << auth.dev_message << std::endl;
        // Run application functions
    } else {
        // Displays exact error (e.g. "Incorrect password.", "HWID mismatch.")
        std::cout << "Access Denied: " << auth.response.message << std::endl;
    }

    return 0;
}

License-Only Verification

std::string key;

if (auth.license(key)) {

} else {
    std::cout << "Verification failed: " << auth.response.message << "\n";

1

Install Requests Dependency

The Python wrapper requires the standard HTTP library requests. Run this command in your terminal/prompt:

pip install requests
2

Initialize and Login

Import LegitAuthApp from auth.py, initialize the application instance, and run login:

from auth import LegitAuthApp

# Initialize application credentials
auth = LegitAuthApp(
    name="YOUR_APP_NAME",
    ownerid="YOUR_OWNER_ID",
    secret="YOUR_SECRET",
    version="1.0"
)

username = input("Username: ")
password = input("Password: ")

# Perform user/pass login
if auth.login(username, password):
    print(f"Login successful! Welcome back.")
    print(f"Developer MOTD: {auth.dev_message}")
else:
    # Prints localized error (e.g. "Incorrect password.", "Account banned.")
    print(f"Login failed: {auth.response.message}")
